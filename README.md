# 🧟‍♂️ UNTOT Zombies

**UNTOT Zombies** ist ein rundenbasiertes 3D-Zombie-Survival-Spiel, inspiriert von klassischen Wave-Shootern. Entwickelt mit **Three.js** und verpackt als **Electron-Desktop-App**, bietet es rasante Action, düstere Atmosphäre und komplexe Spielmechaniken direkt auf deinem PC.

Überlebe so lange wie möglich gegen endlose, immer stärker werdende Wellen von Untoten. Sammle Punkte, schalte neue Bereiche frei, kaufe bessere Waffen und trinke Perks, um zu überleben!

## ✨ Features

*   **Rundenbasiertes Gameplay:** Die Zombies werden mit jeder Runde schneller, zahlreicher und stärker.
*   **Punkte-System:** Verdiene Punkte durch Kills, Headshots und das Reparieren von Barrikaden.
*   **Waffen-Arsenal:** Starte mit einer M1911 und kaufe dir mächtigere Waffen wie die AK47, Schrotflinte, das MG42 oder mystische Energiewaffen.
*   **Mystery Box:** Gib 950 Punkte aus, um eine zufällige Waffe aus der Kiste zu ziehen – mit der Chance auf extrem seltene Ausrüstung.
*   **Perk-Automaten:** Trinke Spezial-Perks, um deine Fähigkeiten zu verbessern:
    *   ❤️ **Juggernaut:** Verdoppelt deine maximale Gesundheit.
    *   ⚡ **Speed Cola:** Halbiert die Nachladezeit.
    *   🔫 **Double Tap:** Erhöht die Feuerrate.
    *   🏃 **Stamin-Up:** Lässt dich schneller sprinten.
*   **Power-Ups:** Zombies lassen gelegentlich Drops fallen (Max Ammo, Insta-Kill, Double Points, Nuke, Carpenter).
*   **Barrikaden:** Repariere Holzbretter an Fenstern, um Zombies aufzuhalten und dir wertvolle Punkte zu sichern.
*   **Immersive 3D-Atmosphäre:** Dynamisches Licht, Nebel (Fog), Post-Processing (Bloom) und detailliertes 3D-Audio mit räumlicher Ortung (Positional Audio).

## 🎮 Steuerung

| Taste | Aktion |
| :--- | :--- |
| **W, A, S, D** | Bewegen |
| **Maus** | Umsehen & Zielen |
| **Linksklick** | Schießen |
| **Rechtsklick (halten)** | Zielen (ADS / Aim Down Sights) |
| **Shift (halten)** | Sprinten |
| **C** | Sliden (während dem Sprinten) |
| **Leertaste** | Springen |
| **R** | Nachladen |
| **E (halten)** | Interagieren (Kaufen, Barrikaden reparieren) |
| **V** | Messern (Nahkampf) |
| **G** | Granate werfen |
| **L** | Taschenlampe an/aus |
| **1 / 2** | Waffen wechseln |
| **F11** | Vollbildmodus umschalten |
| **ESC** | Pause-Menü |

## 🛠️ Technologien

*   **Three.js:** 3D-Rendering, Geometrie, Materialien und Beleuchtung.
*   **Web Audio API / THREE.Audio:** Räumliches 3D-Sounddesign für Waffen, Zombies und Umgebung.
*   **HTML5 / CSS3:** Modernes, responsives Heads-Up-Display (HUD) und Menü-Design.
*   **Electron:** Standalone-Desktop-Anwendung für Windows.

## 🚀 Installation & Starten

Um das Spiel lokal auszuführen oder zu kompilieren, benötigst du [Node.js](https://nodejs.org/).

1. **Repository klonen oder herunterladen**
2. **Abhängigkeiten installieren:**
   ```bash
   npm install
   ```
3. **Spiel im Entwicklermodus starten:**
   ```bash
   npm start
   ```
4. **Spiel als ausführbare `.exe` bauen (Portable):**
   ```bash
   npm run build
   ```
   *Die fertige `.exe` befindet sich danach im Ordner `build/win-unpacked` oder `build/`.*
